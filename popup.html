How to use:

  1. Enter your YouTube Data API v3 key (Required)
  2. Enter the username you want to track (optional)
  3. Choose your preferred comment sorting method
  4. Set toast display duration (for Video Player Time mode)
  5. Click "Save Settings"
  6. Navigate to any YouTube video
  7. The overlay will show comments based on your selected sorting:
    • Top Comments: Most liked comments with/without timestamps
    • Newest Comments: Most recent comments
    • Video Player Time: Live toasts based on video progress

Features:

  • ✅ Real-time comment monitoring
  • ✅ User-specific comment tracking
  • ✅ Automatic timestamp detection (0:19, 1:23, etc.)
  • ✅ Transparent overlay positioned near video
  • ✅ Draggable comment feed
  • ✅ Live notifications
  • ✅ Auto-scroll newest comments
  • ✅ Better comment loading with retries
